CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.

Mobilidade Urbana

Mobilidade urbana é definida como a facilidade de deslocamento das pessoas e bens na cidade, com o objetivo de desenvolver atividades econômicas e sociais no perímetro urbano de cidades, aglomerações urbanas e regiões metropolitanas.
