3 dicas para usar o CSS:


Cada navegador tem pré-definifo alguns atributos, como margins, borders, paddings, entre outros. Para evitar conflitos com a sua folha de estilos, trabalhe sempre com um Css Reset, que é uma classe especial que fará a função de “limpar a casa”.
A não ser que seu dados sejam realmente tabeláveis, não utilize tables! Para a construção de templates e posicionamento de elemento trabalhe com Divs.
Ao definir um estilo, podemos aplica-lo através de elemento com um id especificado ou através da utilização de uma classe. Se um tipo de estilo será aplicado somente a um elemento, prefira trabalhar com ids.
